# Superstore Analysis Dashboard (Tableau)

## Overview
This project presents an interactive business intelligence dashboard developed in Tableau using the Sample Superstore dataset.
The objective was to transform transactional sales data into a user-friendly analytical dashboard that supports business decision-making and performance monitoring.

## Features
### KPI Cards
* Total Sales
* Total Profit
* Number of Orders
* Number of Customers
* Number of Returns
* Profit Margin (%)

### Interactive Metric Selector
Users can switch between:
* Sales
* Profit
* Orders

All dashboard visualizations update dynamically based on the selected metric.

### Visualizations
* Sales / Profit / Orders by State (Map)
* Performance by Customer Segment
* Quarterly and Yearly Trend Analysis
* Top 5 Product Sub-Categories
* Returns Analysis by Regional Managers

### Dashboard Filters
* Order Date
* Ship Mode
* Product

## Tableau Features Used
* Parameters
* Calculated Fields
* Dashboard Actions
* Filters
* Geographic Maps
* Trend Lines
* LOD Expressions (FIXED, INCLUDE)
* Table Calculations
* String Functions

## Business Value
The dashboard enables users to:
* Monitor sales and profitability trends
* Compare customer segments
* Identify top-performing product categories
* Analyze regional performance
* Track product returns
* Explore data interactively through filters and parameters
